
# TaskFlow - To-Do List Application

TaskFlow is a modern productivity-focused To-Do List application built using HTML, CSS, and JavaScript.

## Features

* Add Tasks
* Delete Tasks
* Mark Tasks as Completed
* Task Analytics Dashboard
* Productivity Overview
* Progress Tracking
* Local Storage Support
* Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage API

## Project Preview

TaskFlow helps users organize daily activities through an intuitive and modern interface.
